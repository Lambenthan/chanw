## 5.1 从固定效应到随机效应

固定效应模型假设 $\hat{y}_i = \theta + \varepsilon_i$，所有研究共享同一个真实效应 $\theta$。随机效应模型把这个假设松开：每项研究自己有一个真实效应 $\theta_i$，13 个 $\theta_i$ 不再要求相等，而是从一个总体分布里抽出来的。

<Definition number="5.1" title="随机效应模型">

$$
\hat{y}_i = \theta_i + \varepsilon_i, \quad
\theta_i \sim N(\mu, \tau^2), \quad
\varepsilon_i \sim N(0, v_i).
$$

其中 $\hat{y}_i$ 是第 $i$ 项研究的 $\mathrm{log\ RR}$ 估计，$\theta_i$ 是第 $i$ 项研究的真实效应（每项研究自己一个），$\mu$ 是**平均真实效应**（meta 分析合并的目标）。$\varepsilon_i$ 是抽样误差，方差 $v_i$；$\theta_i$ 自身的分布方差是 $\tau^2$，叫**研究间方差**。$\tau^2 = 0$ 时模型退化为固定效应。

</Definition>

把 $\tau^2$ 翻译成生活语言。假设 13 项 BCG 试验各自的真实效应分别是 $\theta_1, \theta_2, \ldots, \theta_{13}$。固定效应模型说这 13 个数都相等（共有一个 $\theta$）；随机效应模型说这 13 个数不相等，但围绕一个总体均值 $\mu$ 分布，分布的宽度就是 $\tau$。BCG 数据里，伦敦 Hart & Sutherland 1977 的真实效应可能是 $\mathrm{log\ RR} = -1.4$，赤道附近 TPT Madras 1980 的真实效应可能是 $0$，二者本来就不一样。$\tau^2$ 把这种本来就有的差异量化出来，让合并 CI 老老实实变宽。

### 5.1.1 随机效应模型的权重

随机效应模型的权重也按方差倒数计算，但方差包含两部分：研究内的抽样方差 $v_i$ 和研究间的真实方差 $\tau^2$。

<Definition number="5.2" title="随机效应权重">

RE 模型下每项研究的权重为

$$
w_i^{\mathrm{RE}} = \frac{1}{v_i + \tau^2}.
$$

其中 $v_i$ 是第 $i$ 项研究自身的抽样方差（同 FE），$\tau^2$ 是研究间真实效应的方差。合并估计 $\hat{\mu} = \sum_i w_i^{\mathrm{RE}} \hat{y}_i / \sum_i w_i^{\mathrm{RE}}$，方差 $\mathrm{Var}(\hat{\mu}) = 1 / \sum_i w_i^{\mathrm{RE}}$。

</Definition>

把 $\tau^2$ 加进权重的分母会带来两个直接后果。其中一个：所有研究的权重都比 FE 模型小，因为分母变大了，导致合并 CI 比 FE 宽。另一个：原本权重悬殊的研究权重被"摊平"。FE 模型下 TPT Madras 一项就占 41%，到 RE 模型下这个比例会显著缩水，因为 $\tau^2$ 把它的相对优势削弱了。

## 5.2 DerSimonian-Laird 估计量

$\tau^2$ 不是直接观测的，必须从数据里估计。DerSimonian & Laird 1986 提出了最早也是最常用的 $\tau^2$ 矩估计量。

<Definition number="5.3" title="DerSimonian-Laird τ² 估计量">

$$
\hat{\tau}^2_{\mathrm{DL}}
= \max\left\{0,\ \frac{Q - (k - 1)}{\sum_i w_i - \sum_i w_i^2 / \sum_i w_i}\right\}.
$$

其中 $Q$ 是异质性 Q 统计量（第 4 章给出）、$k$ 是研究数、$w_i = 1/v_i$ 是 FE 模型下的逆方差权重。分子 $Q - (k - 1)$ 是观察 Q 超出"在 FE 假设下应有期望 $k-1$"的部分，分母把这个超额量还原到 $\mathrm{log\ RR}$ 平方尺度。如果 $Q \le k - 1$（异质性弱于纯抽样波动），$\tau^2$ 截断为零。

</Definition>

在 BCG 数据上手算一遍。先把第 4 章给出的 13 项研究 IV 权重 $w_i = 1/v_i$ 全部累加：$\sum w_i = 609.70$。再算 $\sum w_i^2 = 94{,}820.58$，所以 $\sum w_i^2 / \sum w_i = 94{,}820.58 / 609.70 = 155.52$。DL 公式分母 $\sum w_i - \sum w_i^2 / \sum w_i = 609.70 - 155.52 = 454.18$。Q 已经在第 4 章算出 $Q = 152.23$，自由度 $k - 1 = 12$，分子 $Q - 12 = 140.23$。代入：

$$
\hat{\tau}^2_{\mathrm{DL}} = \frac{140.23}{454.18} = 0.3088.
$$

分子 140.23 是观察到的研究间散布超出抽样波动的部分，分母 454.18 是把这个超额量还原到 log RR 平方尺度的换算因子。

```r
library(metafor)
d  <- read.csv("data/bcg.csv")
es <- escalc(measure = "RR", ai = tpos, bi = tneg,
             ci = cpos, di = cneg, data = d)

# 手算 DL τ² 的中间量
w       <- 1 / es$vi
sum_w   <- sum(w)              # 609.70
sum_w2  <- sum(w^2)            # 94820.58
denom   <- sum_w - sum_w2 / sum_w   # 454.18
Q       <- rma(yi, vi, data = es, method = "EE")$QE   # 152.23
tau2_DL <- (Q - 12) / denom
cat("tau^2 (DL, 手算) =", round(tau2_DL, 4), "\n")

# DerSimonian-Laird τ² 估计（与手算应当一致）
re_dl <- rma(yi, vi, data = es, method = "DL")
print(re_dl)
```

RE-DL 给出 $\hat{\tau}^2_{\mathrm{DL}} = 0.3088$，$\hat{\tau} = 0.556$。合并 $\widehat{\mathrm{logRR}} = -0.7141$，标准误 0.1787，95% CI $[-1.0644,\ -0.3638]$，对应 $\mathrm{RR} = 0.49$，95% CI $[0.35,\ 0.70]$。

与 FE-IV 的 $\mathrm{RR} = 0.65$ 相比，RE-DL 的合并 RR 偏小（保护效应更强），CI 宽度从 FE 的 $[0.60,\ 0.70]$ 变成 $[0.35,\ 0.70]$。这种 CI 大幅变宽是合理的：随机效应承认研究之间存在真实差异，CI 必须把这种差异纳入不确定性。

DL 是矩估计，它要求 $Q$ 的期望值在 FE 模型下等于 $k - 1$，超出 $k - 1$ 的部分就归因于研究间方差。这个推导不需要假设 $\theta_i$ 服从正态，是一种"分布无关"的估计。代价是 DL 在小研究数（$k < 20$）和高异质性下**偏低**：它会低估 $\tau^2$，进而高估权重精度，让 CI 偏窄。

## 5.3 REML 估计

REML（Restricted Maximum Likelihood，限制性最大似然）是基于正态假设的最大似然估计，理论上效率更高，在小研究数下偏倚比 DL 小。Cochrane Handbook 与 metafor 包都把 REML 作为默认 $\tau^2$ 估计量。

<Definition number="5.4" title="REML τ² 估计">

在假设 $\theta_i \sim N(\mu, \tau^2)$ 与 $\hat{y}_i \mid \theta_i \sim N(\theta_i, v_i)$ 下，REML 通过对"抹去 $\mu$ 后的边际似然"求极值得到 $\tau^2$ 估计。形式上没有解析解，由迭代算法求解，metafor 默认使用 Fisher scoring 算法。

</Definition>

REML 的优势在小样本下尤为明显。DL 的矩估计在 $k$ 小的时候 $Q$ 本身的方差就不可忽略，DL 估计波动很大。REML 用整个似然函数的形状信息，对 $\tau^2$ 的估计更稳。代价是 REML 假设 $\theta_i$ 服从正态，DL 不假设。实务上正态假设的违反在 $k$ 较大时影响有限。

```r
re_reml <- rma(yi, vi, data = es, method = "REML")
print(re_reml)
```

REML 给出 $\hat{\tau}^2_{\mathrm{REML}} = 0.3132$，与 DL 的 0.3088 几乎一致。合并 $\widehat{\mathrm{logRR}} = -0.7145$，95% CI $[-1.0669,\ -0.3622]$，$\mathrm{RR} = 0.489$，95% CI $[0.34,\ 0.70]$。BCG 数据上 DL 与 REML 结果几乎重合，说明 $k = 13$ 已经足够让两种估计趋同。在更小的 $k$（如 $k = 5$）下，二者会拉开。

---

## 5.4 Hartung-Knapp 调整

DL 与 REML 都用大样本正态近似算合并 CI：把 $\hat{\mu}$ 的方差当作正态分布的方差，乘以 1.96 算 95% CI 半宽。这个近似在 $k$ 大时合理，$k$ 少时偏窄。Hartung & Knapp 2001 提出用 $t$ 分布替代正态、自由度 $k - 1$，让 CI 更稳健。

<Definition number="5.5" title="Hartung-Knapp 调整">

HK 调整修改两个东西。其一，把合并估计的标准误从 $\sqrt{1/\sum_i w_i^{\mathrm{RE}}}$ 替换为

$$
\mathrm{SE}_{\mathrm{HK}}
= \sqrt{\frac{\sum_i w_i^{\mathrm{RE}} (\hat{y}_i - \hat{\mu})^2}
              {(k - 1) \sum_i w_i^{\mathrm{RE}}}},
$$

其中 $w_i^{\mathrm{RE}}$ 是 RE 权重，$\hat{\mu}$ 是 RE 合并均值。其二，置信区间用 $t_{k - 1}$ 分布的临界值替代 1.96：

$$
\mathrm{CI}_{95\%}
= \hat{\mu} \pm t_{k - 1, 0.975} \cdot \mathrm{SE}_{\mathrm{HK}}.
$$

metafor 通过参数 `test = "knha"` 启用 HK 调整。

</Definition>

HK 调整从两处把 CI 拉宽。一处是用每项研究偏离合并均值的实际散布替换理论方差，让 CI 反映实际看到的散布，而不是理论应有的散布。另一处是把 $z_{0.975} = 1.96$ 换成 $t_{k - 1, 0.975}$。当 $k = 13$，$t_{12, 0.975} = 2.179$，比 1.96 大约 11%；$k = 5$ 时 $t_{4, 0.975} = 2.776$，比 1.96 大约 42%。研究数越少，HK 调整把 CI 拉得越宽，避免小样本下的过度自信。

```r
re_hk <- rma(yi, vi, data = es, method = "REML", test = "knha")
print(re_hk)
```

HK 调整后合并 $\widehat{\mathrm{logRR}} = -0.7145$（点估计与 REML 完全相同），标准误 0.1808（略高于 REML 的 0.1798），95% CI $[-1.1084,\ -0.3206]$，$\mathrm{RR} = 0.489$，95% CI $[0.33,\ 0.73]$。$p$ 值用 $t$ 分布算，$p = 0.0019$（仍显著）。

Cochrane Handbook 自 2019 年起推荐随机效应模型默认使用 REML + HK 组合，尤其在 $k < 20$ 时。本书后续的随机效应主结果均采用这组配置。

## 5.5 预测区间 PI

随机效应模型有一个比 CI 更直接对应临床决策的指标：**预测区间**（prediction interval，简称 PI）。CI 描述合并均值的不确定性，PI 描述未来一项新研究的真实效应可能在哪里。在异质性显著时 PI 比 CI 重要得多。

<Definition number="5.6" title="预测区间">

$$
\mathrm{PI}_{95\%}
= \hat{\mu} \pm t_{k - 2, 0.975} \cdot
  \sqrt{\hat{\tau}^2 + \mathrm{SE}(\hat{\mu})^2}.
$$

其中 $\hat{\mu}$ 是 RE 合并均值，$\hat{\tau}^2$ 是研究间方差估计，$\mathrm{SE}(\hat{\mu})$ 是合并均值的标准误。PI 比 CI 宽，因为根号里把 $\hat{\tau}^2$（未来研究自身的真实异质性）和 $\mathrm{SE}(\hat{\mu})^2$（合并估计的不确定性）两部分加在一起。**自由度 $k - 2$ 而非 HK CI 的 $k - 1$**：PI 公式根号里多估了一个 $\tau^2$ 参数，按 IntHout-Higgins-Tovey 2016 的推导多扣一个自由度。metafor 的 `predict()` 函数对带 `test = "knha"` 的对象会自动用对应自由度计算 CI 与 PI。

</Definition>

PI 与 CI 的差别直接影响临床判断。CI 告诉你的是"全球 BCG 的平均效应"；PI 告诉你的是"在某一个具体地区开展 BCG 项目，预期效应的范围"。如果 PI 跨过零线，意味着未来一项研究完全可能给出无效甚至反向的结果，临床推广就要谨慎。

```r
pi_hk <- predict(re_hk)
cat("合并 RR =", round(exp(pi_hk$pred), 3),
    "  95% CI =", round(exp(pi_hk$ci.lb), 3), "to",
                  round(exp(pi_hk$ci.ub), 3), "\n")
cat("95% PI  = [", round(exp(pi_hk$pi.lb), 3), ",",
                   round(exp(pi_hk$pi.ub), 3), "]\n")
```

REML + HK 给出合并 $\mathrm{RR} = 0.489$，95% CI $[0.33,\ 0.73]$，95% PI $[0.136,\ 1.763]$。CI 不跨过 1，"全球平均 BCG 有保护效应"这个结论在 5% 错误率下成立。但 PI 跨过 1，意味着未来在某个新地区开展的 BCG 试验可能给出从 $\mathrm{RR} = 0.14$（强保护，把风险压到原来的 14%）到 $\mathrm{RR} = 1.76$（反向，比对照组风险高 76%）的任何结果。这就是 BCG 数据给出的临床信息：BCG 平均有效，但在不同地区效果可能截然不同。第 7 章的纬度 meta 回归会回答这种"具体地区效应"如何用协变量解释。

## 5.6 Forest plot 带预测区间

随机效应 forest plot 通常在合并菱形下方加一根额外的横线代表 PI，方便临床读者直接读出"未来研究的可能效应范围"。metafor 通过参数 `addpred = TRUE` 自动添加这一行。

</FullWidthSection>

<Pitfall title="把 PI 当 CI 解读 / 异质性极高时报告 RE 合并均值">

随机效应模型有两个常被混淆的雷区。第一是**把 PI 当 CI 解读**。论文里读到"95% interval = [0.13, 1.76]"，不知道是 CI 还是 PI 就草率引用，会得到完全相反的结论。CI 描述合并均值的不确定性，PI 描述未来研究的不确定性。报告时必须明确标注。

第二是**在异质性极高时报告 RE 合并均值**。当 $I^2 > 85\%$，不同研究的真实效应很可能本质上不同，合并的 $\mu$ 可能对应不到任何实际场景。BCG 数据 $I^2 = 92\%$，单一合并 $\mathrm{RR} = 0.49$ 只是统计意义上的全球平均，不能直接解读为"在我所在的地区 BCG 把风险降低 51%"。

**稳健做法**：配合第 7 章的亚组与 meta 回归，解释异质性来源；报告 PI 时明确标注"未来研究的可能效应范围"。

</Pitfall>

## 5.7 本章小结

三种 RE 估计（DL、REML、REML+HK）给出几乎相同的点估计 $\mathrm{RR} = 0.49$，CI 宽度从 DL 的 $[0.35, 0.70]$ 一路扩到 REML+HK 的 $[0.33, 0.73]$。比 FE-IV 的 $[0.60, 0.70]$ 显著更宽是合理的：RE 模型承认研究本来就不一样，CI 必须把这种差异纳入不确定性。

但合并 $\mathrm{RR} = 0.49$ 与 PI $[0.14, 1.76]$ 跨度极大的反差说明，**异质性本身需要被正面量化**。第 6 章就用 $Q$、$I^2$、$\tau^2$ 这一组指标系统刻画异质性，PI 不再重复定义，按本章 Definition 5.6 直接引用。
