## 3.1 空模型的方程与符号含义

<Definition number="3.1" title="空模型与随机截距">

对学生 $i$ 嵌套在班级 $k$ 里的两层数据，最简单的空模型写作

$$
y_{ik} = \beta_0 + u_k + e_{ik}, \quad u_k \sim N(0, \tau^2), \quad e_{ik} \sim N(0, \sigma^2),
$$

其中 $y_{ik}$ 是班级 $k$ 里第 $i$ 名学生的数学分数，$\beta_0$ 是全样本"总均值"对应的固定截距，$u_k$ 是班级 $k$ 相对总均值的偏差，服从均值为 0、方差为 $\tau^2$ 的正态分布；$e_{ik}$ 是该学生相对班级均值的个体波动，服从均值为 0、方差为 $\sigma^2$ 的正态分布；$u_k$ 与 $e_{ik}$ 互相独立。"空模型"指右边没有任何解释变量，只用截距加随机效应来描述结果。

</Definition>

把这条公式翻成话：每名学生的分数等于一个总基线 $\beta_0$，加上他所在班级的"班级偏差" $u_k$，再加上他个人在班里的"个体波动" $e_{ik}$。$\beta_0 = 486.30$ 就是全样本数学均值的估计；$u_k$ 是某个班相对这条基线高几分还是低几分；$e_{ik}$ 是某名学生相对班级均值的余下偏离。

```r
library(tidyverse)
library(lme4)
library(lmerTest)
library(here)

d <- read_csv(here::here("data", "star_k.csv"), show_col_types = FALSE) |>
  filter(!is.na(math), !is.na(cltype), !is.na(tch), !is.na(sch))

m0 <- lmer(math ~ 1 + (1 | tch), data = d, REML = TRUE)
summary(m0)
# Linear mixed model fit by REML. t-tests use Satterthwaite's method
# Formula: math ~ 1 + (1 | tch)
#
# Random effects:
#  Groups   Name        Variance Std.Dev.
#  tch      (Intercept)  689     26.25
#  Residual             1595     39.94
# Number of obs: 5786, groups:  tch, 337
#
# Fixed effects:
#             Estimate Std. Error      df t value Pr(>|t|)
# (Intercept)  486.30      2.49   333.6  195.3    <2e-16

VarCorr(m0)                          # 随机效应方差分量
ranef_class <- ranef(m0)$tch         # 337 个班的 BLUP
```

完整脚本见 [code/chap03.R](/books/hlm/code/chap03.R)。

这份输出包含三块关键信息。第一块是 Random effects：班级间方差 $\tau^2 = 689$，对应班级间标准差 $\sqrt{689} = 26.25$；班级内方差 $\sigma^2 = 1{,}595$，对应学生层标准差 $\sqrt{1{,}595} = 39.94$。班级层 26 分相当于"两个班的均值平均差 26 分"这种数量级；学生层 40 分相当于"班里同学之间的分数高低差 40 分"这种数量级。

第二块是 Fixed effects：固定截距 $\beta_0 = 486.30$，标准误 2.49，意思是"考虑了班级嵌套之后，对全样本数学均值的估计是 486.3，误差棒是 2.5 分"。这跟简单平均得到的 487 分非常接近，但 SE 比朴素 OLS 的 0.65 大了将近 4 倍，这是嵌套结构对均值估计本身把握度的折损。

第三块是底下的 `Number of obs: 5786, groups: tch, 337`，这一行是 `lmer` 在帮你确认数据结构没读错：5,786 名学生分散在 337 个班级里。

## 3.2 BLUP 与收缩：班均值向全样本均值的回归

<Definition number="3.2" title="BLUP">

对随机截距模型 $y_{ik} = \beta_0 + u_k + e_{ik}$，每个班级 $k$ 的随机截距 $u_k$ 用一个带收缩权重的公式来估，而非直接采用"班级均值减全样本均值"：

$$
\hat{u}_k = \frac{n_k \tau^2}{n_k \tau^2 + \sigma^2} \bigl(\bar{y}_k - \beta_0\bigr).
$$

其中 $n_k$ 是班级 $k$ 的人数，$\bar{y}_k$ 是该班的原始均值，$\beta_0$ 是全样本固定截距，$\tau^2$ 与 $\sigma^2$ 是两层方差。这个估计被称为最优线性无偏预测量（best linear unbiased predictor，BLUP），由 Henderson 1950 年代在动物育种里推导出来。

</Definition>

光看公式有点抽象，把它代成具体数字就很直观。STAR 里某个班 12 名学生原始均值 545 分，全样本均值 486.3 分，原始偏差是 $545 - 486.3 = 58.7$ 分。代入 BLUP 公式，权重是

$$
\frac{12 \times 689}{12 \times 689 + 1{,}595} = \frac{8{,}268}{9{,}863} = 0.838.
$$

其中 12 是班级人数，689 是班级间方差，1,595 是班级内方差。BLUP 估计是 $0.838 \times 58.7 = 49.2$，比原始偏差 58.7 缩了将近十分。这个缩水现象就是**收缩**（shrinkage）。

这样估的道理可以从抽样的角度看。在 79 所学校的 337 个班里抽 12 个孩子算班均值，如果这 12 个孩子里恰好坐了几个考试当天发挥极好的学霸，原始均值会被拉得离谱地高。BLUP 的设计思路是："样本量小的班、班均值特别极端的班，更可能是抽样偶然导致的，向全样本均值靠回去更靠谱；样本量大、信息充足的班，自己的均值就足够可信，不用调多少。"权重 $n_k \tau^2 / (n_k \tau^2 + \sigma^2)$ 恰好实现这个设计：$n_k$ 大权重接近 1，几乎不动；$n_k$ 小权重远离 1，往全样本均值靠。

回到 STAR 的全样本数字：337 个班的原始均值标准差是 28.59，BLUP 标准差是 24.29，整体收缩了 $(28.59 - 24.29)/28.59 = 15\%$。BLUP 的范围从 $-71.04$ 到 $+85.16$，开根号一下，$\sqrt{\tau^2} = \sqrt{689} = 26.25$，跟 BLUP SD = 24.29 接近：理论上 BLUP SD 应该略小于 $\sqrt{\tau^2}$，因为收缩让方差缩小了一点。

<Subtext>图 3.1 · 班级 BLUP 收缩示意图。</Subtext>

这张图把 337 个班按原始均值排好，每个班画一对点，灰线连接同一班的两种估计。两端最极端的几个班（左端最低、右端最高）向中间收缩得最显眼，中间一带的班几乎没动。这跟 BLUP 公式给出的逻辑完全对应：极端值更可能是抽样偶然，应该被收缩回去。

BLUP 这个概念是 1950 年 Henderson 在康奈尔大学农学院提出来的，原本是给牛奶产量遗传值的预测用的。育种学家面对的问题跟教育研究面对的问题数学形式相同：从有限观测里估计每头母牛、每个班级的"真实潜力"，要兼顾"自身观测"和"群体先验"。

## 3.3 REML vs ML：两种估计方法在多层模型里的分工

`lmer` 默认用 REML（restricted maximum likelihood，限制性极大似然）拟合。为什么不用更"标准"的 ML（极大似然）？这背后有个微妙但重要的偏差问题。

ML 估计方差分量时把所有观测代进似然函数一起最大化，结果是固定效应跟方差分量"互相影响"。具体后果是 $\tau^2$ 跟 $\sigma^2$ 会被系统性低估，幅度跟参数个数有关。Patterson 与 Thompson 1971 年在 *Biometrika* 上提出 REML，思路是"先用一个变换把固定效应那一块从似然里去掉，再对剩下的部分做极大似然"。这样估出来的方差分量是无偏（或近似无偏）的。

代价是 REML 的似然值不能用来比较不同固定效应结构的模型，因为变换本身依赖固定效应设定。如果想用似然比检验比较两个模型的固定效应（比如"加 cltype 比不加 cltype 好不好"），必须切到 ML。

具体到 STAR 的空模型，REML 和 ML 给出的方差分量差别非常小：

| 参数              | REML   | ML     |
|:------------------|:------:|:------:|
| $\tau^2$（班级间方差）   | 689    | 686    |
| $\sigma^2$（班级内方差） | 1,595  | 1,595  |
| $\beta_0$（固定截距）    | 486.30 | 486.30 |

<Subtext>表 3.1 · 空模型 REML 与 ML 估计对比。</Subtext>

样本量 5,786、班级数 337 的情况下，两种方法差别小到可以忽略。但样本量小的时候差别会放大。Bates 等人写 `lme4` 时把 REML 设为默认，正是为了让用户在大多数场景下拿到的是无偏估计；只在做模型比较时显式切到 `REML = FALSE`。

<Pitfall title="把 ML 估计当成无偏">

有些教材、有些早期 SAS 代码默认用 ML 估方差分量，给出的 $\tau^2$ 系统性偏低。当用 ML 算 ICC，分子 $\tau^2$ 偏低、分母也偏低，比值不一定怎么变；但当 $\tau^2$ 用来做后续计算（比如设计效应、BLUP 收缩权重），就会传递误差。

**诊断方法**：检查 `lmer` 调用是否显式设置了 `REML = TRUE` 或者用了默认值。如果代码里写了 `REML = FALSE`，看是不是只在做似然比检验之前临时切的。

**记忆方法**：估方差用 REML，比固定效应用 ML，最后报告时再切回 REML。`lmerTest` 的 `anova` 在两个 `lmer` 模型之间做 LRT 时会自动 refit 成 ML，不用自己切。

</Pitfall>

## 3.4 似然比检验：空模型相对朴素 OLS 是否值得

读完空模型的所有数字之后，要回答的最后一个问题是："加上 `(1|tch)` 这个随机截距，相比纯 $y = \beta_0 + e_i$ 的 OLS，到底好多少？"答案要靠似然比检验给。

似然比检验比较两个嵌套模型的对数似然，统计量

$$
\text{LR} = -2 \bigl(\ell_{\text{simple}} - \ell_{\text{complex}}\bigr).
$$

其中 $\ell$ 是各模型的对数似然，简单模型嵌套在复杂模型里。在原假设"复杂模型不比简单模型好"下，LR 服从自由度等于参数差的卡方分布。

把朴素 OLS（仅截距）跟空模型放进 `anova` 比较。两者参数差是 1（多了一个 $\tau^2$），但 $\tau^2$ 在原假设下卡在边界 0 上，标准卡方分布不再适用，要用 $0.5 \chi^2_0 + 0.5 \chi^2_1$ 这种"边界混合"分布，Self 与 Liang 1987 年专门处理过这个边界问题。`lme4` 跟 `RLRsim` 都内置了正确的检验程序，常规用法直接用 `anova` 比较 `lmer` 跟 `lm`，得到 LR = 1,355.15、$p < 2.2 \times 10^{-16}$。

LR = 1,355 在 5,786 个学生 + 337 个班这个样本规模下属于合理量级。ICC = 30.2% 这种强嵌套结构下，纯 OLS 模型抓不住任何班级层信息，似然被显著拉低。LR = 1,355 的具体数值不是重点，它显著超过任何合理的卡方临界值，多层结构必须建模。这条结论第 2 章已经从"标准误膨胀 2.38 倍"给出了一次，本节用似然比再验证一次，两条独立证据指向同一个判断。

## 3.5 从空模型到学生层协变量

本章把空模型从头到尾解了一遍。读者现在应该能复述这套逻辑：空模型方程 $y_{ik} = \beta_0 + u_k + e_{ik}$ 把每名学生分数拆成总基线、班级偏差、个体波动；`lmer` 给出的输出包含 $\beta_0 = 486.30$、$\tau^2 = 689$、$\sigma^2 = 1{,}595$ 三块；337 个班的 $u_k$ 用 BLUP 估计，估出来比原始班均值整体收缩 15%，因为极端班更可能是抽样偶然；REML 给方差分量、ML 给固定效应似然比检验，`lme4` 默认 REML 不用自己切；空模型相对纯 OLS 的 LR 高达 1,355，强嵌套结构必须建模。

但本章只用了"空模型"这一个最简单的设定。STAR 数据里还有大量被忽略的信息：学生的性别、族裔、是否享受免费午餐、教师的工龄与学位、学校类型。这些变量要么是学生层的（每个学生不一样），要么是班级或学校层的（同班同校的学生共享）。下一章先处理学生层协变量。会加性别、族裔、SES 三个变量，看 $\sigma^2$ 能被压下去多少（学生层变量解释的是学生层方差），看 $\tau^2$ 会不会被影响（教育研究里这块通常变化很小）。还会引入组均值中心化（centering within cluster）这个方法，讲清同一个变量 SES 在"个体层"和"班级层"两个层面的效应可以完全不同，不分开看会把两者混合成单一系数。
