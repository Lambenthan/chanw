但如果回去翻第 1 章末尾那张图，会注意到一个一直没被处理的事实：79 所学校的数学均值跨度有 124 分，这部分跨校差异在前面所有模型里被悄悄塞进了"班级层"。当一所学校自己的均值偏低 30 分，这所学校里的每一个班级都会在 $u_k$ 估计里继承这 30 分的偏差，让"班级方差"看起来比实际大。本章要做的事就是把学校层从班级层里分出来，跑一个完整的三层模型，看看分出来之后小班效应会变成什么样。

## 8.1 三层嵌套的结构：学生、班级、学校

为了让"为什么要加学校层"不只是一个统计学口号，先用一个生活类比把三层嵌套讲清。

想象一户家族里几代同堂。最底下是**孩子**，每个孩子有自己的脾气与天赋；中间一代是**父母**，同一对父母带的几个孩子共享家庭的教养方式、餐桌习惯、作息节奏；最上面是**祖辈**，同一个祖父祖母往下分出去的几对父母，又共享更上层的家训、家族财力、家族口碑。如果想解释"这家的孩子为什么学习成绩好"，把每个孩子当独立样本来看会漏掉父母层面的相似性，把父母当独立样本看又会漏掉祖辈带来的家族影响。

STAR 数据的三层结构正是这副族谱。**学生**对应孩子，每名学生有自己的个体差异；**班级**对应父母，同一班的 17 个学生共享一位老师、一间教室、一种学习氛围；**学校**对应祖辈，同一所学校的若干个班共享一个校长、一份生源、一个社区。前面所有章节用的 `(1|tch)` 只承认了"父母层面的共享"，把祖辈这一层默认为不存在。

如果一所学校所在的社区里平均家庭背景较好、师资稳定、教学传统强，这所学校的所有班级都会在分数上整体偏高。如果不把这部分"学校层共同优势"独立出来，它会被错算成班级层的"班际波动"，让人误以为"班际差异"比实际更大。

## 8.2 嵌套随机效应与三层模型形式

<Definition number="8.1" title="嵌套随机效应">

对一份三层嵌套数据（学生 $i$ 在班级 $k$ 在学校 $j$），结果 $y_{ijk}$ 的最简三层模型为

$$
y_{ijk} = \beta_0 + v_j + u_{jk} + e_{ijk},
$$

其中三个随机项相互独立：$v_j \sim N(0,\, \tau^2_{\text{sch}})$ 是学校 $j$ 的随机截距偏差，$u_{jk} \sim N(0,\, \tau^2_{\text{class}})$ 是嵌套在学校 $j$ 里的班级 $k$ 的随机截距偏差，$e_{ijk} \sim N(0,\, \sigma^2)$ 是学生层残差。随机截距 $u_{jk}$ 嵌套在 $v_j$ 之内，意味着不同学校里的"班级 1"是完全不同的班级、不共用同一个 $u$。这是"嵌套"区别于"交叉"的核心要求。

</Definition>

公式里的三层结构对应着族谱里的三代。$\beta_0$ 是全样本期望均值，所有学校、所有班级、所有学生的"基准线"；$v_j$ 把学校 $j$ 拉离这条基准线，所有这所学校里的班级与学生都共享这一份偏差；$u_{jk}$ 在学校 $j$ 内部再把班级 $k$ 拉离学校均值，本班所有学生再共享这一份偏差；$e_{ijk}$ 把学生 $i$ 在班级均值之上做最后一次个人波动。

为什么要用"嵌套"而不是"交叉"？因为 STAR 里的"班级 1"在 79 所学校里指的是 79 个完全不同的班级，没有任何理由让它们共享同一个班级随机效应。这与"星期一在所有学校都是星期一"的交叉结构不同——星期一这个"组"在不同学校之间确实是同一回事。`lme4` 的 `(1|sch/tch)` 语法等价于 `(1|sch) + (1|sch:tch)`，它告诉模型"班级 $k$ 只在学校 $j$ 里有意义"，自动把跨校的同名班级当成不同的随机单元。

<Definition number="8.2" title="三层 ICC 分解">

对三层空模型 $y_{ijk} = \beta_0 + v_j + u_{jk} + e_{ijk}$，总方差被拆成

$$
\mathrm{Var}(y_{ijk}) = \tau^2_{\text{sch}} + \tau^2_{\text{class}} + \sigma^2.
$$

对应三个组内相关系数：

$$
\mathrm{ICC}_{\text{sch}}
  = \frac{\tau^2_{\text{sch}}}{\tau^2_{\text{sch}} + \tau^2_{\text{class}} + \sigma^2},
\quad
\mathrm{ICC}_{\text{class}|\text{sch}}
  = \frac{\tau^2_{\text{class}}}{\tau^2_{\text{sch}} + \tau^2_{\text{class}} + \sigma^2},
\quad
\mathrm{ICC}_{\text{within}}
  = \frac{\sigma^2}{\tau^2_{\text{sch}} + \tau^2_{\text{class}} + \sigma^2}.
$$

其中 $\mathrm{ICC}_{\text{sch}}$ 是同一所学校两名学生的预期相关性，$\mathrm{ICC}_{\text{class}|\text{sch}}$ 是同一所学校同一班级两名学生在学校层效应已扣除之后剩余的相关性，$\mathrm{ICC}_{\text{within}}$ 是纯学生层差异占总方差的比例。三者相加等于 1。

</Definition>

## 8.3 三层空模型：把方差拆成学校 / 班级 / 学生三块

跑空模型只需要把 `(1|tch)` 换成 `(1|sch/tch)`，剩下完全一样。

```r
library(tidyverse)
library(lme4)
library(lmerTest)
library(here)

d <- read_csv(here::here("data", "star_k.csv"), show_col_types = FALSE) |>
  filter(!is.na(math), !is.na(cltype), !is.na(tch), !is.na(sch),
         !is.na(sx), !is.na(eth), !is.na(ses)) |>
  mutate(
    sex_F  = ifelse(sx == "F", 1, 0),
    eth_B  = ifelse(eth == "B", 1, 0),
    ses_F  = ifelse(ses == "F", 1, 0),
    cltype = factor(cltype, levels = c("reg", "reg+A", "small"))
  )

# 三层空模型：班级嵌套在学校里
m0_3l <- lmer(math ~ 1 + (1 | sch/tch), data = d, REML = TRUE)
as.data.frame(VarCorr(m0_3l))
#         grp        var1  vcov
# 1   tch:sch (Intercept)   283
# 2       sch (Intercept)   365
# 3  Residual        <NA>  1492
```

三层空模型把 STAR 数学分数的总方差约 2,140 拆成三块。**学校层**方差 $\tau^2_{\text{sch}} = 365$，对应 $\mathrm{ICC}_{\text{sch}} = 365 / 2{,}140 = 16.9\%$。**班级层**方差（学校已扣除）$\tau^2_{\text{class}} = 283$，对应 $\mathrm{ICC}_{\text{class}|\text{sch}} = 283 / 2{,}140 = 13.2\%$。**学生层**残差方差 $\sigma^2 = 1{,}598$，对应 $\mathrm{ICC}_{\text{within}} = 1{,}598 / 2{,}140 = 69.6\%$（三层空模型数字与全协变量模型数字略有不同，此处为空模型）。三个 ICC 之和恰好为 1。STAR 里的"嵌套相关"被这三块各分一份，学校层占了 17%，与班级层的 13.2% 量级相当。

![三层空模型方差分解。学校层 365 占 16.9%，班级层 283 占 13.2%，学生层 1,598 占 69.6%](/books/hlm/figure/chap08_three_level_pie.png)

<Subtext>图 8.1 · 三层空模型方差分解饼图：学校层与班级层量级相当，学生层内方差占七成。</Subtext>

把这三个 ICC 跟第 3 章的二层空模型对比一下就能看出区别。二层空模型给出的"班级层方差"是 689，对应 ICC 30.2%。这 689 在三层分解之后，分裂成了 $\tau^2_{\text{sch}} = 365$ 和 $\tau^2_{\text{class}} = 283$ 两块，加起来恰好 648，跟二层的 689 非常接近。这不是巧合：如果不让模型看到"学校"这个层级，所有学校层的系统性差异只能被堆进班级层。**二层的"班级方差"实际上是"学校 + 班级"两层方差的混合体**。

如果一所学校的所有班级都比全样本均值高 19 分（$\sqrt{365} = 19.10$），二层模型只能把这 19 分塞进每个班的 $u_k$ 里，让"班际差异"看起来更大。三层模型能识别出"这 19 分是同一所学校共享的"，把它从班级层挪到学校层，班级层只剩"学校已扣除之后班际还剩下多少差异"。

## 8.4 学校层随机截距 BLUP 的分布

三层模型顺便给出每所学校的随机截距 BLUP $\hat{v}_j$，相当于"扣掉班级与学生层效应之后，这所学校相对全样本均值高出多少分"。把 79 所学校按 BLUP 排序画出来，就能直观看到学校层的实际差异。

![79 所学校的随机截距 BLUP。横轴是学校排名，纵轴是扣除班级与学生层效应后学校相对全样本均值的偏差](/books/hlm/figure/chap08_school_blup.png)

<Subtext>图 8.2 · 学校 BLUP 排序图：BLUP 从约 −30 到 +30 横跨 60 分，远大于小班效应的 8 分。</Subtext>

读这张图的方法是看横向的扩展度。BLUP 从大约 $-30$ 到 $+30$ 横跨 60 分，意味着最差的学校"基准线"比最好的学校低 60 分左右。这 60 分跟"小班 vs 常班"的 8 分差距相比非常大——**学校之间的系统性差异远大于处理效应本身**。如果不把学校层扣出来，处理效应估计就要在这 60 分的噪声里挣扎。

为什么这些 BLUP 不集中在零附近？因为 $\sqrt{\tau^2_{\text{sch}}} = \sqrt{365} = 19.10$，所以单所学校的 BLUP 大致在 $-2 \times 19 = -38$ 到 $+38$ 之间散开（按正态假设的 95% 范围）。图上看到的散布幅度跟这个理论值非常吻合。

## 8.5 加上协变量与处理变量：三层模型的完整版

把第 4 到第 7 章的所有协变量装进三层模型，得到本书目前为止最完整的形式：

$$
\texttt{math} \sim \texttt{cltype} + \texttt{sex\_F} + \texttt{eth\_B}
                  + \texttt{ses\_F} + (1 \mid \texttt{sch}/\texttt{tch}).
$$

```r
m_3l_full <- lmer(math ~ cltype + sex_F + eth_B + ses_F +
                    (1 | sch/tch),
                  data = d, REML = TRUE)
summary(m_3l_full)$coefficients
#                Estimate Std. Error      df t value  Pr(>|t|)
# (Intercept)    491.94      2.99   158.7  164.69  <2e-16
# cltypereg+A      0.00      2.85   239.9    0.00   1.000
# cltypesmall      8.84      2.63   233.6    3.36   0.001
# sex_F            6.12      0.97  5429.9    6.32   3e-10
# eth_B          -10.89      2.34   126.3   -4.66   8e-06
# ses_F          -19.86      1.20  5544.5  -16.49  <2e-16

# 二层版本（同一份协变量，没有 sch）作为对比
m_2l_full <- lmer(math ~ cltype + sex_F + eth_B + ses_F + (1 | tch),
                  data = d, REML = TRUE)
summary(m_2l_full)$coefficients["cltypesmall", ]
#  Estimate Std. Error         df    t value   Pr(>|t|)
#     7.68       3.63       321.1     2.12      0.035
```

小班相对常班的处理效应在三层模型里是 **8.84 分**，标准误 **2.63**，$p = 0.001$。跟二层模型的 7.68 分（SE = 3.63，$p = 0.035$）相比，点估计**更大、标准误反而更小**。这是本章最反直觉的结果：加进一层随机效应，理论上在拟合更复杂的模型，反而把"小班效应"的不确定度压低了。

解释这件事的关键在于把"学校层"看清。二层模型把学校层差异塞进班级层 $u_k$，让 $\tau^2_{\text{class}} = 638$ 变成"两层差异之和"，这个膨胀的方差直接进入小班效应的标准误估计。三层模型把它分开：班级层只剩 $\tau^2_{\text{class}} = 283$，跨校系统性差异被 $v_j$ 单独承担。处理变量 `cltype` 是在班级层变化的，它的标准误主要受班级层方差影响，班级层方差变小，处理效应的 SE 自然下降。

这里要再仔细看一下方差分解的对应关系。三层模型 + 全协变量下，$\tau^2_{\text{sch}} = 365$、$\tau^2_{\text{class}} = 283$、两者相加 $648$。二层模型 + 全协变量下，$\tau^2_{\text{class}} = 638$。两个数字相差不到 2%。这是一条干净的加和关系：

$$
\underbrace{\tau^2_{\text{class, 2L}}}_{638}
  \;\approx\;
\underbrace{\tau^2_{\text{class, 3L}}}_{283}
  + \underbrace{\tau^2_{\text{sch, 3L}}}_{365}.
$$

二层模型不是"算错"了，它只是没有把学校层独立出来，所有学校层差异只好寄居在班级层方差里。加和关系告诉我们**两层方差与三层方差总量相等，区别在分配方式**。

小班效应的 SE 由"班级层方差占整体的比例"决定（因为处理是班级层变量）。二层模型的班级层方差是 638，三层模型的班级层方差是 283，分子从 638 变成 283，SE 自然从 3.63 降到 2.63。三层模型让 SE 下降的原因，是"它把原本错算到班级层的方差挪走，让真正属于班级层的方差归位"，而不是"模型变复杂了所以拟合得更紧"。

## 8.6 LRT 检验：三层 vs 二层的模型对比

加上学校层这一层值不值？方差分解里 $\tau^2_{\text{sch}} = 365$ 已经占总方差 17%，是不是显著到必须独立成一层，还要做正式的统计检验。对"是否引入额外随机效应"的检验，标准做法是**似然比检验**（LRT）。两个模型一个是简化版（只有班级随机截距），一个是完整版（再加学校随机截距），用 ML 拟合两个模型，把对数似然之差乘以 2，对照 $\chi^2$ 分布查临界值。

```r
m_2l_ml <- lmer(math ~ cltype + sex_F + eth_B + ses_F + (1 | tch),
                data = d, REML = FALSE)
m_3l_ml <- lmer(math ~ cltype + sex_F + eth_B + ses_F + (1 | sch/tch),
                data = d, REML = FALSE)
anova(m_2l_ml, m_3l_ml)
#           npar   AIC   BIC  logLik deviance  Chisq Df Pr(>Chisq)
# m_2l_ml      7 59218 59272 -29602    59204
# m_3l_ml      8 59130 59190 -29557    59114  90.68  1   <2.2e-16
```

LRT 给出 $\chi^2 = 90.68$，自由度为 1（多估了一个 $\tau^2_{\text{sch}}$），$p < 2.2 \times 10^{-16}$，拒绝"学校层方差为零"的零假设。AIC 也从 59,218 降到 59,130，下降 **89** 个单位。按经验法则 $\Delta\mathrm{AIC} > 10$ 已支持复杂模型，89 显著超过这个门槛。STAR 数据里的学校层不是边缘效应，是模型必须诚实承认的层级。

LRT 在"检验方差分量是否为零"时有一个微妙之处：方差不能为负，零假设是参数取在参数空间边界上，标准的 $\chi^2_1$ 临界值是保守的，真实的零分布是 $\frac{1}{2}\chi^2_0 + \frac{1}{2}\chi^2_1$ 的混合分布（Self 与 Liang 1987；Stram 与 Lee 1994）。本数据 $\chi^2 = 90.68$ 远高于任何合理临界值，这点细节不影响结论；但读者将来跑临界值附近的 LRT 时要记得这件事。

<Pitfall title="把三层结构错写成二层的后果：班级方差伪膨胀 + 处理效应 SE 偏大">

如果数据有学校层但只用 `(1|tch)` 拟合，模型会把学校层的系统性差异强行塞进班级层随机截距。最直接的后果是**班级层方差被伪膨胀**——本数据里 $\tau^2_{\text{class}}$ 从真实的 283 涨到 638，看上去班级之间的差异比实际大一倍多。膨胀的方差又会传染给**处理效应标准误**：处理变量 `cltype` 在班级层变化，它的 SE 受班级层方差影响，本数据二层 SE 是 3.63、三层 SE 是 2.63，差距正来自这部分错算。除此之外，**学校层 BLUP 拿不到**也是一笔损失：想知道"哪所学校特别好、哪所特别差"，二层模型完全没有这个估计，只有 `(1|sch/tch)` 才会在 `ranef` 里返回 `$sch` 这一项。

**诊断方法**：跑空模型时同时拟合二层与三层，比较 $\tau^2_{\text{class, 2L}}$ 跟 $\tau^2_{\text{class, 3L}} + \tau^2_{\text{sch, 3L}}$。如果两者接近、且 $\tau^2_{\text{sch, 3L}}$ 占总方差超过 5%，就必须用三层模型。

</Pitfall>

<Subtext>表 8.1 · 累积方法对比（截至第 8 章）。读这张表的关键在两处：small β 从 7.68 跳到 8.84，SE 同时从 3.63 降到 2.63——点估计变大、不确定度变小，处理效应的统计支持显著加强。</Subtext>

到本章为止，本书的模型已经包含学生层协变量、班级层处理变量与学校层随机截距，形式上是教育学多层建模文献里非常完整的标准三层模型。本数据的结论是：小班相对常班 +8.84 分，95% CI 大致 $[3.7,\, 14.0]$，$p = 0.001$，比第 5 章二层估计的 7.68 分更大、不确定度更小。

但模型给出"$p = 0.001$"这个结论之前，需要做最后一道工序：检查模型本身是不是合身。残差有没有系统性偏离假设？正态性在三层都成立吗？某些班级或学校的 BLUP 是不是大到能单独驱动整个估计？这些问题构成第 9 章的内容。

## 8.8 本章知识地图
