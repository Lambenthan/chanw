## 10.1 终极对照表：silicon sample 的 8 个体检项

| 诊断项 | 指标 | 本书数字 | 合格阈值 | 失败信号 | 修复方向 |
|:-------|:-----|:--------:|:--------:|:---------|:---------|
| 边际分布 | TVD polviews                | 0.30          | $<0.15$       | 分布形状失真       | 用 Llama 3 / 减 RLHF      |
| 边际分布 | TVD partyid                 | 0.27          | $<0.15$       | 独立派归零         | 多 LLM 集成               |
| 个体匹配 | ACC polviews                | 32.3%         | $>40\%$       | 弱于 modal baseline | 加入 logistic baseline    |
| 个体匹配 | MAE partyid                 | 2.04          | $<1.0$        | 远端错占主导       | 重设计 prompt             |

<Subtext>表 10.1 · Silicon sample 完整体检表。</Subtext>

把本书数字一行行读，silicon sample 在 100 persona × 3 题这份数据上不通过任何一项合格阈值。这一结论并非说 silicon sampling 无用，而是说在严格审稿要求下，本书演示的 Claude silicon sample 不能直接用于发表论文。读者在自己数据上做 silicon sampling 应该按这张体检表逐项核查。

## 10.2 硅基社会学派与人格因果学派的对照

本书是"硅基社会学派"第一本，与"人格因果学派"（YangmingTrajectory / SushiTrajectory / ZengGuofanTrajectory / GuYanwuTrajectory / PersonalityCausality_Meta 五本）构成一对方法学平行。

| 维度       | 人格因果学派                                     | 硅基社会学派                                     |
|:-----------|:-------------------------------------------------|:-------------------------------------------------|
| 研究对象   | 真实历史人物（阳明 / 苏轼 / 曾国藩 / 顾炎武）   | 大语言模型（Claude / GPT-4 / Llama）             |
| 数据形态   | 文本（诗 / 文 / 信 / 录）                        | 问卷（人口学 + 态度题）                          |
| 主方法     | ITS / 散度 / 断点 / 合成控制 / FE               | TVD / Cohen's $\kappa$ / subgroup bias           |
| 研究问题   | 外生冲击如何改变人格                             | demographic prior 如何造出 attitude              |
| 单被试     | 一个历史人物                                     | 一个 LLM 实例                                    |
| 可复现     | 文本固定可重读                                   | 模型版本滚动更新                                 |

<Subtext>表 10.2 · 两派的对照。</Subtext>

两派的相同点：单被试研究、纵向 / 横向变异分解、需要诚实交代外部效度边界。两派的差异在研究对象的**客体性**——历史人物的文本是 immutable 客体，LLM 的输出是 ephemeral 客体（模型版本一换就变）。这一差异让硅基社会学派的可复现挑战远大于人格因果学派。

## 10.3 本书未覆盖的议题

silicon sampling 的方法学发展很快，本书 100 persona × 3 题的小演示无法覆盖以下议题：

**Fine-tuning silicon**：把 LLM 在 GSS 历史数据上 fine-tune，看 silicon 是否能接近真实分布。Argyle 2024 working paper 显示 fine-tuning 能把 TVD 从 0.30 压到 0.10，但 fine-tuning 是否过拟合特定波次的 GSS 是开放问题。

**Multi-lingual silicon**：silicon sampling 在英语之外的语言（中文 CGSS、德文 GESIS）上是否还成立。社区报告 silicon 在英语之外 LLM 的 prompt sensitivity 显著更高。

**Conjoint experiment silicon**：让 LLM 扮演消费者做 product choice。Brand 2023 NBER WP 是这一方向的开端，本书没覆盖。

**Policy intervention 模拟**：用 silicon 模拟"如果给 X 群体推送 Y 政策"。这是 silicon 最有 policy 价值的应用方向，也是 Bisbee 2024 反复警告"不要轻易用"的方向。

**Network silicon**：让多个 silicon respondent 互动（如 Park et al. 2023 Stanford Generative Agents 项目）。这一方向涉及 agent-to-agent 交互动力学，本书完全未涉及。

## 10.4 未来 2--3 年的学派研究方向

把 silicon social science 在中国学界落地，未来 2--3 年值得做的研究方向有：

**方向一**：把本书的诊断框架移植到 CGSS / CFPS / WVS 中文样本，测 Claude / Qwen / DeepSeek 在中文人口学上的 silicon fidelity。这是中文 silicon sampling 的第一篇方法学论文应有的形态。

**方向二**：拿 silicon 与人格因果学派的四案例对照——让 Claude 扮演阳明 / 苏轼 / 曾国藩 / 顾炎武，看 silicon 给出的态度文本与真实文本的散度，把 silicon sample 推到**历史人物模拟**任务，与人格因果学派的 corpus 数据直接比对。

**方向三**：构造 silicon 的 *adversarial benchmark*——专门设计 LLM 一定会答错的 demographic case（如双重身份的少数族裔），把 silicon 的 failure mode 系统化。

**方向四**：silicon 与传统 missing data imputation 的对比。silicon 本质是用 LLM 做 imputation，传统方法（MICE、IPW、PMM）已有几十年文献。比较两者在 GSS 缺失数据重建上的表现。

这四个方向覆盖方法学诊断 / 历史模拟 / 对抗设计 / 与经典方法对比四类问题，正好填本书没占的空间。

## 10.5 致研究生与青年研究者

silicon sampling 在 2023--2025 年是社会科学 LLM 应用最热的子领域，国内中文学界刚开始跟进。一个有效的入门路径是：读 Argyle 2023 原文，跑本书的 100 persona × 3 题 demo（约 30 分钟），跑通后扩到 1000 persona × 10 题，再做跨 LLM 复现，最后投 *Sociological Methods & Research* / *中国社会科学评价* / *社会学评论* 等期刊。整套流程 6--12 个月可走完，是研究生论文的合理量级选题。
