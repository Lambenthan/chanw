<ChapterAudience>

把四案例内部时段散度演化放在同一张表里，看话语变化幅度的跨案例规律。用双外部对照空间看四案例的演化轨迹形态，引出「事件 + 时间窗」的双重判断：同样事件在不同案例上的话语效应窗口期不同。

</ChapterAudience>

<IntroParagraph>第 2 章的 ITS 给的是 *单一 treatment 上的局部效应*，但有些案例的话语变化不集中在 treatment 附近，而是在 treatment 之后多年才显现，或分散在多个晚期节点。散度分析能补这一缺口，它给的是 *时段间的整体话语距离*，不预设事件位置。</IntroParagraph>

|:---------|:---------------------------------------------------------------------------------------------------------------------------|

| 苏轼     | P1 早年（1056–1070）/ P2 反新法（1071–1078）/ P3 黄州（1080–1084）/ P4 元祐起复（1085–1093）/ P5 惠州（1094–1096）/ P6 儋州与北归（1097–1101） |

## 3.2 时段间过渡的 JS 散度（内部话语变化幅度）

把每个案例的相邻时段过渡按 JS 散度计算，数值越大表示话语变化越剧烈：

| 过渡      | 阳明              | 苏轼              | 曾国藩            | 顾炎武            |
|:----------|:------------------|:------------------|:------------------|:------------------|
| P1 → P2  | 0.21（廷杖期，大） | 0.08（反新法，中） | 0.08（创湘军，中） | 0.18（北上，大）  |
| P2 → P3  | 0.13（南赣）      | 0.07（黄州）      | 0.06（安庆）      | 0.11（山陕）      |
| P3 → P4  | 0.18（宁王）      | 0.23（元祐）      | 0.10（洋务）      | ——                |
| P4 → P5  | 0.09（致良知）    | **0.44（惠州）**   | 0.15（暮年）      | ——                |
| P5 → P6  | 0.16（南安）      | 0.23（儋州）      | ——                | ——                |

<Subtext>表 3.2 · 四案例相邻时段的 JS 散度。</Subtext>

四案例的话语变化幅度分布有显著差异。

**阳明 + 顾炎武**：最大过渡集中在 P1 → P2，即 *冲击发生时段*。阳明 1506 廷杖前后 JS = 0.21，顾炎武 1660 北上前后 JS = 0.18，两者结构相似：冲击立即引发话语剧变。

**苏轼**：最大过渡在 P4 → P5（JS = 0.44，远超其他过渡），即 *1093 哲宗亲政导致再贬惠州的党争翻案期*。1079 乌台诗案（P2 → P3）的过渡 JS 仅 0.07，反而是 5 次过渡中最小的。这条结果与第 2 章的 ITS 结论一致：苏轼的话语断裂落在元祐起复之后的党争反转，而非乌台。

**曾国藩**：5 次过渡中最大值仅 0.15，远小于阳明、苏轼、顾炎武的峰值。这说明曾国藩的话语演化是 *渐进型*，没有单一时段构成剧烈断裂。这与曾国藩家书第 1 章 ITS 在 D2 D8 等内省维度上不显著的结果呼应：曾国藩走的是 *稳定累积*，没有单点的 self-reframing 转折。

**顾炎武**：顾炎武只有 2 次过渡可观察，数据点少，P1 → P2 的 0.18 是初步估计，后续 iteration 应扩展时段切分到 P3 → P4（晚年）。但与四案例对照下，顾炎武 P1 → P2 的 0.18 与阳明 P1 → P2 的 0.21 在同一量级，验证了国变型与廷狱型在 *话语剧变时机* 上的相似性。

<Pitfall title="为什么不直接比 4 案例的最大 JS 散度">

最大 JS 散度依赖时段切分的颗粒度：阳明 6 时段、苏轼 6 时段、曾国藩 5 时段、顾炎武 3 时段，时段数差异让最大值不可比。更稳的做法是看 *话语剧变的 timing*，把 *是否落在 treatment 当年的过渡上* 作为跨案例对比指标，而把 *绝对幅度* 放在次要位置。按这个指标，阳明、顾炎武 = 「冲击当时立即剧变」；苏轼 = 「冲击后 13 年才剧变」；曾国藩 = 「始终不剧变，稳定累积」。这种四案例的 *话语断裂 timing* 与第 2 章 ITS 的 *D2 D8 维度变化时间* 高度一致，是互相印证的双方法学结果。

</Pitfall>

| 案例     | 异质轴对照             | 同质轴对照             |
|:---------|:-----------------------|:-----------------------|
| 阳明     | 朱熹（理学正统）       | ——（单对照）            |
| 苏轼     | 王安石（政治异质）     | 黄庭坚（苏门同质）     |
| 曾国藩   | 李鸿章（实业派）       | 左宗棠（同袍后期分歧） |
| 顾炎武   | 王夫之（隐居型遗民）   | 黄宗羲（士林反清型遗民） |

| 时段          | JS(vs 李鸿章) | JS(vs 左宗棠) | 相对位置                                |
|:--------------|--------------:|--------------:|:----------------------------------------|
| P1 翰林       |         0.433 |         0.510 | 翰林期距双方都最远（内省 vs 奏牍）       |
| P2 创湘军     |         0.326 |         0.354 | 进入军务话语，距双方都靠近              |

<Subtext>表 3.5 · 曾国藩相对双外部对照的二维 JS 距离。Parity: JS(李鸿章, 左宗棠) = 0.051。</Subtext>

曾国藩的 parity = 0.051 非常低（李左两人奏牍互相距离极近，因为同为公文体），这是曾国藩项目诚实交代的方法学缺陷：*双对照实质退化为单对照*，两位对照人物代表不了异质轴。后续 iteration 应当替换其中一位为非奏牍体（倭仁读书录或左宗棠诗钞）。即使在 parity 较低的双对照下，曾国藩的 P1 翰林期距双方都最远（0.43 / 0.51）仍是一个稳健信号，说明翰林期家书与奏牍体在话语上差异明显，反映出曾国藩私人（家书）与公务（奏牍）的两套语言体系。

### 3.3.3 顾炎武的二维轨迹

顾炎武相对王夫之 + 黄宗羲的 JS 距离：顾炎武整体 JS(vs 王) = 0.192，JS(vs 黄) = 0.246，parity = 0.224（这是四案例中 parity 最高的，真正构成有意义的二维空间）。由于顾炎武时段切分只有 3 段，详细的二维轨迹演化数据不足，这条留待后续语料扩充。

## 3.4 跨案例话语演化形态的对比

把四案例的内部散度模式与双对照空间合起来，可以归纳出三种话语演化形态。

**形态 A（戏剧性形态）**：单次大幅过渡，然后趋稳。阳明（P1 → P2 廷杖期 0.21 后逐渐回归）+ 顾炎武（P1 → P2 北上期 0.18 后稳定）属于此类。

**形态 B（延迟剧变形态）**：早期平稳，晚期突然剧变。苏轼（P1–P3 都在 0.07–0.13 区间，P4 → P5 突然 0.44）属于此类。

**形态 C（稳定累积形态）**：始终无大过渡，渐进演化。曾国藩（所有过渡 $< 0.15$）属于此类。

三种形态对应不同的心理学机制。形态 A 是 *急性应激重组*：外生冲击触发完整人格响应，随后系统重新稳定。形态 B 是 *累积负担断裂*：多次冲击叠加，单次冲击不足以剧变，累积到某临界点（苏轼 1094 党争翻案）才崩塌。形态 C 是 *预先准备稳定*：个体冲击前已有完整内省工夫（曾国藩日课十二条），外生冲击不需要触发新的内省，只需要在已有框架内吸收。

这三种形态的存在表明，**「外生冲击 → 人格变化」没有单一机制**，响应模式依赖个体的人格基线状态。这与第 2 章 D8 三教融合的方向分裂结论（依赖思想空间）在结构上同源：人格变化的方向 + 节奏都依赖个体的冲击前状态。
