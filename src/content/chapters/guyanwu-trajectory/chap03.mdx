用 JS 散度比较顾炎武 vs 王夫之 vs 黄宗羲三人话语分布；给出 JS(顾, 王) = 0.1920，JS(顾, 黄) = 0.2461，JS(王, 黄) = 0.2244 三组距离；解读三遗民共有的"易代遗民语汇"基底与各自学术特性；与曾国藩双对照同质性（parity = 0.05）对比，三遗民对照独立性更好。

<IntroParagraph>第 2 章用 ITS 估顾炎武 1660 北上的局部效应。本章用 JS 散度比较顾炎武与王夫之、黄宗羲三人的整体话语分布，看顾炎武在 *易代遗民共有话语* 之上的剩余个体特性。</IntroParagraph>

<Subtext>表 3.1 · 三遗民数据规模（殆知阁公版整理本）。</Subtext>

三人字数数量级相近（50–80 万），比较具有可比性。

## 3.2 跨作者 JS 散度

| 对照                       | JS 散度    |
|:---------------------------|-----------:|
| **JS(顾炎武, 王夫之)**     | **0.1920** |
| JS(顾炎武, 黄宗羲)         | 0.2461     |
| JS(王夫之, 黄宗羲)         | 0.2244     |

<Subtext>表 3.2 · 三遗民两两 JS 散度（基于 9 主题 56 个核心概念）。</Subtext>

三对 JS 距离都在 0.19–0.25 区间，显著大于曾国藩项目李鸿章 vs 左宗棠的 0.0510，也大于苏轼项目王安石 vs 黄庭坚的 0.17。这条 *对照独立性更好* 让顾炎武项目的双外部对照设计在方法学上比曾国藩项目更稳健。

**顾炎武距王夫之最近（JS = 0.1920）**：这条距离最小有两条解释。一是两人都偏 *经史考据 + 政治哲学*，学术语汇高度同源；二是两人都长期不出（王隐居衡山 30 年，顾北游山东山西陕西 25 年），与黄宗羲的 *讲学型遗民* 学术语境不同。

**顾炎武距黄宗羲最远（JS = 0.2461）**：黄宗羲 *明儒学案* 是学案体，大量引用前人学者言论，文体上与顾炎武日知录的札记 + 议论体差异最大。这条距离差异主要来自体裁本身，而非思想立场。

**王夫之距黄宗羲（JS = 0.2244）** 居中，说明三人话语分布形成一个 *近等边三角形*，没有两人特别近、第三人特别远的极端结构。

## 3.3 三作者 top 概念差异

| 顾炎武 | 王夫之 | 黄宗羲 |
|:-------|:-------|:-------|
| 书     | 知     | 知     |
| 传     | 民     | 学     |
| 山     | 亡     | 明     |
| 史     | 将     | 问     |
| 知     | 兵     | 书     |

<Subtext>表 3.3 · 三作者 top 5 概念词（从 56 个核心概念中）。</Subtext>

三作者的 top 概念词共同体现易代遗民共有的 *知*（知识探求）高频。但各自的次高词显示学术路径差异。

**顾炎武 *书 / 传 / 山 / 史***：偏 *文献考据 + 北游山水*，反映他北游 25 年访古寻碑的实地考据风格。

**王夫之 *民 / 亡 / 将 / 兵***：偏 *政治哲学 + 国亡反思*，反映他 *读通鉴论* 借古代史评近世政治的写作角度。

**黄宗羲 *学 / 明 / 问***：偏 *学术史 + 师承梳理*，反映他 *明儒学案* 建立中国第一部学案体史学的工作。

三作者经历同样的 1644 甲申之变，但学术语汇的侧重点完全不同。这条 *易代共历 / 路径分化* 给"人格因果"学派提供了重要观察：**外生冲击的话语后果在不同个体上沿不同方向分化，没有统一的"易代遗民话语模板"**。

<Pitfall title="为什么三人话语差异这么大">

易代是 *结构性外生冲击*，但每个人的反应是 *基于既有学术根基的延展*。顾炎武入清前已在江南做经史考据（1639 始撰天下郡国利病书），入清后延续这条路径转向北游访古；王夫之入清前精研周易宋明理学，入清后延续这条转向通鉴史评；黄宗羲入清前从父黄尊素习东林党学问，入清后延续这条转向学案史。三条路径在入清前已分化，入清后只是 *延续 + 深化*，不是 *易代触发新方向*。

这条结论对"人格因果"学派的方法学有重要含义：**外生冲击的"反思方向"由个体 pre 期已有的学术 / 性格基底决定，冲击本身只决定"反思强度"与"延迟长度"，不决定"反思维度"**。

</Pitfall>

## 3.4 双外部对照的方法学优势

| 案例         | 对照 A             | 对照 B             | Parity JS(A,B)        |
|:-------------|:-------------------|:-------------------|:----------------------|
| 阳明         | 朱熹（单对照）     | —                  | —                     |
| 苏轼         | 王安石（新党）     | 黄庭坚（苏门）     | 0.17                  |
| 曾国藩       | 李鸿章（奏牍）     | 左宗棠（奏牍）     | 0.05（同质过高）      |
| **顾炎武**   | **王夫之（隐居型遗民）** | **黄宗羲（讲学型遗民）** | **0.22**       |

<Subtext>表 3.4 · 四案例双外部对照的同质性对比。</Subtext>

顾炎武的 Parity JS = 0.22 是四案例中最高的，意味着双对照内部最独立，顾炎武距双对照的距离差异（JS(顾, 王) = 0.19 vs JS(顾, 黄) = 0.25）主要反映 *顾炎武自身的话语特性*，不是 *双对照内部混淆*。这条独立性让本章的双对照分析比曾国藩项目更稳健。
