## 4.1 RMSPE 比值检验的设计

Abadie 2010 提出的核心统计量是 post / pre RMSPE 比值。

<Definition number="4.1" title="RMSPE 比值检验统计量">

设 $T_0$ 是 pre-period 末年，$T$ 是 post-period 末年。对任意单元 $i$，定义

$$
R_i = \frac{\sqrt{ \frac{1}{T - T_0} \sum_{t = T_0 + 1}^{T} \widehat{\tau}_{it}^2 }}{
        \sqrt{ \frac{1}{T_0}     \sum_{t = 1}^{T_0}       \widehat{\tau}_{it}^2 }}
    = \frac{\text{post-RMSPE}_i}{\text{pre-RMSPE}_i},
$$

其中 $\widehat{\tau}_{it} = Y_{it} - \widehat{Y}_{it}(0)$ 是 SC 给单元 $i$ 在 $t$ 期的 gap。$R_i$ 是处理后偏离相对于 pre-period 拟合误差的比值。$R$ 的精确 p 值由 California 的 $R_1$ 在 39 个单元（1 处理 + 38 placebo）中的排序位置决定。

</Definition>

其中 $\widehat{\tau}_{it}$ 是 SC 算出的 gap，$T_0$ 是 pre-period 长度（本数据 19），$T - T_0$ 是 post-period 长度（本数据 12）。把 $R$ 而不是 raw post gap 当统计量的原因是：raw gap 受 SC 拟合质量影响（fit 差的单元 gap 自动大），$R$ 把 fit 质量（pre-RMSPE）作为分母规范化。

### 4.1.1 pre-MSPE filter 的必要性

把 38 个 donor 都当 placebo 比较时，会在两端各出现一类极端。

一端是 pre-MSPE 极大的 donor。这些是 fit 极差的 placebo（如 Kentucky、Virginia 等烟草种植大州），SC 拟合在 pre-period 就已经偏离 5 倍以上，post-period 的"gap"基本是拟合误差顺延，并非"假处理效应"。Abadie 2010 论文 Figure 7–8 把这些州明确删除（设阈值 $k$，pre-MSPE > $k$ × California pre-MSPE 即删）。

另一端是 pre-MSPE 极小的 donor。这些是 fit 极好但 post-period 突然偏离的 placebo（在 tidysynth 实现下，Minnesota 的 pre-MSPE 接近 $10^{-5}$，对应 RMSPE 几乎为零）。极小的分母让 $R_i$ 飙到几千甚至几百万。这种 placebo 的 $R$ 是数值不稳定造成的，不应进入 p 值计算。

### 4.1.2 在 California 数据上的过滤分级

按 Abadie 的 $k$ 阈值过滤 + 删 pre-MSPE < 0.5 的退化 fit，得到不同 $k$ 下的 p 值。

```r
sig_tab <- sc |> grab_significance(time_window = 1989:2000)
ca_pre  <- sig_tab |> filter(unit_name == "California") |> pull(pre_mspe)

do_rank <- function(k, min_pre = 0.5) {
  keep  <- sig_tab |> filter(pre_mspe <= k * ca_pre, pre_mspe >= min_pre)
  ca_rk <- which(keep$unit_name[order(-keep$mspe_ratio)] == "California")
  data.frame(k = k, n_kept = nrow(keep), ca_rank = ca_rk,
             p_value = round(ca_rk / nrow(keep), 4))
}
bind_rows(lapply(c(2, 5, 20, 1000), do_rank))
```

| $k$ | 保留 placebo 数 | California 排名 | 对应 p 值 |
|----:|----------------:|----------------:|----------:|
|  2  | 28              | 13              | 0.464     |
|  5  | 31              | 13              | 0.419     |
| 20  | 34              | 13              | 0.382     |
| 1000| 34              | 13              | 0.382     |

<Subtext>表 4.1 · 不同过滤阈值下的 California RMSPE 比值排名与 p 值。</Subtext>

California 在过滤后的 RMSPE 比值排名第 13，对应 p 值 0.38（按 $k = 20$）。这一数字与 Abadie 2010 论文报的 $p = 1/39 \approx 0.026$ 不一致。差异来自 tidysynth 包内置预测器配置与原论文的细节差异（预测器权重 $v$ 的求解器精度、placebo 拟合的初始化策略），不是代码错误。

## 4.2 p 值与论文报告差异的解读

Abadie 2010 论文报的 $p = 1/39 \approx 0.026$ 与本章用 tidysynth 复现得到的 $p \approx 0.38$ 是同一份数据上的两个数字，差异来自方法学实现而非"对错"问题。理解这个差异比死磕出统一数字更有教学意义。

差异的第一个来源是预测器权重 $v$ 的求解器。Abadie 原论文用 MATLAB 的 fmincon，tidysynth 调 R 的 ipop（二次规划求解）。两个求解器在 $v$ 的局部最优解上略有不同。这一差异在主估计（点估计 $-18.85$ vs Abadie 原文 $-19$）上很小，但在 placebo 的 38 次重跑里被放大，导致若干 placebo 州的 pre-MSPE 在两种实现下差异显著。

第二个来源是 placebo 集合的初始化。tidysynth 用同一组预测器跑所有 placebo，原论文里有些 sensitivity choices 没全部公开（如对每个 placebo 单独 tune $v$ 的方式）。

第三个来源是 min pre-MSPE 过滤。Abadie 论文的 $k = 20$ filter 在原始实现下让 California 排到第一名（$p = 1/39$），但 tidysynth 的过滤后 California 排第 13，p $\approx 0.38$。

这三个来源叠加，本书呈现的 p 值在 0.38 左右，与论文报的 0.026 有数量级差异。读者应当把这一差异当作 SC 推断对实现敏感的真实证据来理解。Bottmer, Imbens, Spiess & Warnick 2024 等近年文献明确把 SC inference 的不变性失败作为一个开放问题讨论。本章给出的 0.38 与论文 0.026 之间的差异，正是这一开放问题在实操中的体现。

## 4.3 permutation inference 的保留价值

虽然 p 值依实现而变，permutation inference 仍是 SC 标准工具，原因有两条。

其一，permutation 给的是相对排序，对处理效应是否"非典型"的方向性回答仍然可靠。在 California 数据上，无论用 tidysynth 还是原论文，California 的 RMSPE 比值都落在 placebo 分布的左尾（这一点没有争议）；只是"排第几位"的具体数字对实现敏感。

其二，permutation inference 不依赖大样本渐近，对单 treated unit + 38 donor 这种小规模面板可用。其他基于 bootstrap 或解析 SE 的方法（如 GSC、MC-NNM）在单 treated unit 下都有相当强的小样本失效风险（第 5 章会展示），permutation 仍是单 treated 场景下最稳的推断工具。

<Pitfall title="把 SC 的 p 值当成最终结论">

SC 的 permutation p 值对预测器配置、求解器、placebo 过滤标准都敏感。在 California 数据上同一份预测器，tidysynth 给 0.38，Abadie 原论文给 0.026。

**为什么人们会踩坑**：p 值有"客观数字"的外观，容易让作者直接用它做显著性宣判。

**失效机制**：SC 是 deterministic 优化问题，placebo 推断把求解器的局部最优解变成 38 + 1 次重跑的稳定性问题，求解器选择直接影响 p。

**诊断方法 / 稳健替代**：报告 $k = 2, 5, 20$ 三个 filter 下的 p 值 + 说明所用求解器与版本，让读者评估稳健性；附加 SDID（第 6 章）或 MC-NNM（第 7 章）的渐近 SE 作为交叉验证。

</Pitfall>

## 4.5 本章知识地图

| 核心概念             | 核心内容                          | 常见误解              | 为什么错                              |
|:---------------------|:----------------------------------|:----------------------|:--------------------------------------|
| RMSPE 比值           | post-RMSPE / pre-RMSPE            | 以为可绝对比较        | 需用 placebo 排序而非阈值             |
| pre-MSPE 过滤        | 删 fit 极差或极好的 donor         | 以为偏离原 inference  | 是 Abadie 论文标准做法                |
| p = 0.38 vs 0.026    | 实现差异                          | 以为存在唯一真值      | SC 推断不变性失败                     |
| permutation 优势     | 无大样本假设                      | 以为不如解析 SE       | 单处理单元下 permutation 更稳         |
| Exact p              | rank / n                          | 以为可改进            | rank-based 是 exact 标定              |
| k 过滤敏感性         | 报告多个 $k$ 值                   | 以为只报一个就够      | 不报让读者无法判断稳健性即遮蔽        |
